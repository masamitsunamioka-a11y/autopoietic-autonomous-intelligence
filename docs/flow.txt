Detailed Runtime Signal Flow
============================

Async boundary: All callbacks execute on Nucleus thread pool (Ch.9,12).
Recursive:      Lines marked [R1]-[R4] re-enter a prior node in the tree.


CEN — Central Executive Network
================================

Stimulus.input(String)
└── Transducer.transduce(Stimulus)                          Ch.21
    └── Impulse(signal, area=null)
        └── Thalamus.relay(Impulse)  ◄─────────────────── [R4 re-entry]
            ├── Transmitter.transmit(impulse, Projection)
            │   ├── Encoder.encode(impulse, Thalamus)       relay.md
            │   ├── DiffusicService.call(signal)            <- LLM
            │   └── Decoder.decode(raw, Projection)
            │       └── Projection{reasoning, confidence, area}
            │
            └── Nucleus.integrate(projection, callback)     Ch.9,12
                └── [thread pool]
                    ├── Impulse(signal, area=Projection.area)
                    └── Cortex.respond(Impulse)  ◄─────── [R1-R3 re-entry]
                        ├── Transmitter.transmit(impulse, Decision)
                        │   ├── Encoder.encode(impulse, Cortex)     perception.md
                        │   │   ├── input:     impulse.signal()
                        │   │   ├── episode:   Episode.retrieve()
                        │   │   ├── knowledge: Knowledge.retrieve()
                        │   │   ├── self:      impulse.area()
                        │   │   ├── neurons:   area.neurons()
                        │   │   └── effectors: area.effectors()
                        │   ├── DiffusicService.call(signal)        <- LLM
                        │   └── Decoder.decode(raw, Decision)
                        │       └── Decision{reasoning, confidence, process,
                        │                    response?, effector?, area?}
                        │
                        └── Nucleus.integrate(decision, callback)   Ch.9,12
                            └── [thread pool]
                                └── Process.handle(impulse, decision)
                                    │
                                    ├── VOCALIZE
                                    │   ├── Episode.encode(Trace)
                                    │   └── Event<Percept>.fire()   <- CDI
                                    │       └── Salience.release()  @Observes
                                    │
                                    ├── FIRE
                                    │   ├── Effector.fire(knowledge) Ch.35-36
                                    │   ├── Knowledge.encode(results.*)
                                    │   └── Cortex.respond(impulse) ─── [R1] recurse
                                    │
                                    ├── POTENTIATE
                                    │   ├── Plasticity.potentiate(impulse)
                                    │   │   └── [see Plasticity — Potentiation]
                                    │   └── Cortex.respond(impulse) ─── [R2] recurse
                                    │
                                    ├── PROJECT                     Ch.20

                                    │
                                    └── INHIBIT
                                        ├── Episode.encode(Trace)
                                        └── Event<Percept>.fire()   <- CDI
                                            └── Salience.release()  @Observes


DMN — Default Mode Network
============================

@PostConstruct
├── scheduleFire()           every 10-30s    Ch.62
└── scheduleConsolidation()  every 30-60s

Default.fire()
├── if Salience.isOriented() -> return      [CEN busy, skip]
├── Transmitter.transmit(null, Fluctuation)
│   ├── Encoder.encode(null, Default)       default.md
│   │   ├── episode:   Episode.retrieve()
│   │   └── knowledge: Knowledge.retrieve()
│   ├── DiffusicService.call(signal)        <- LLM
│   └── Decoder.decode(raw, Fluctuation)
│       └── Fluctuation{reasoning, confidence, aroused,
│                vocalize, area?, signal?}
│
└── Nucleus.integrate(fluctuation, callback)       Ch.9,12
    └── [thread pool]
        ├── if !fluctuation.aroused() -> return
        ├── introspect(fluctuation, area)
        │   ├── Episode.encode(Trace "[DMN]")
        │   └── Event<Percept>.fire()       <- CDI
        ├── if fluctuation.vocalize()
        │   └── Salience.orient()           [latch armed]
        └── Thalamus.relay(Impulse) ──────────── [R4] recurse into CEN
            └── ... -> Event<Percept>
                └── Salience.release()      -> latch.countDown()

Default.consolidate()                       TODO: Sleep cycle (Ch.65-67)
└── Plasticity.prune()                      should drive pruning/consolidation,
    └── [see Plasticity — Pruning]          not DMN. Introduce sleep concept.


SN — Salience Network
======================

[CEN stimulus arrives]
└── Salience.orient()                       Seeley 2007; Ch.63
    ├── latch = new CountDownLatch(1)
    └── oriented = true
        └── [DMN suppressed: isOriented() -> skip]

[CEN completes -> VOCALIZE/INHIBIT]
└── Event<Percept>.fire()
    └── Salience.release(@Observes)
        ├── oriented = false
        └── latch.countDown()
            └── [DMN resumes on next schedule]

Salience.await()                            [Engineering]
└── latch.await()                           [blocks caller]


Plasticity — Potentiation
==========================

Plasticity.potentiate(Impulse)
├── Transmitter.transmit(impulse, Potentiation)
│   ├── Encoder.encode(impulse, Plasticity)     potentiation.md
│   │   ├── input:     impulse.signal()
│   │   ├── episode:   Episode.retrieve()
│   │   ├── knowledge: Knowledge.retrieve()
│   │   └── self:      impulse.area()
│   ├── DiffusicService.call(signal)            <- LLM
│   └── Decoder.decode(raw, Potentiation)
│       └── Potentiation{reasoning, confidence,
│                        newTuning, newAreas[],
│                        newNeurons[], newEffectors[]}
│
└── Nucleus.integrate(potentiation, callback)   Ch.9,12
    └── [thread pool]
        ├── reinforce(potentiation, area)
        │   └── if newTuning -> Area.store(updated tuning)
        └── sprout(potentiation)
            ├── newEffectors -> effectorRepository.store()
            ├── newNeurons  -> neuronRepository.store()
            └── newAreas    -> sanitize() + areaRepository.store()
                └── sanitize: filter by exists()


Plasticity — Pruning
=====================

Plasticity.prune()
├── Transmitter.transmit(null, Pruning)
│   ├── Encoder.encode(null, Plasticity)        pruning.md
│   ├── DiffusicService.call(signal)            <- LLM
│   └── Decoder.decode(raw, Pruning)
│       └── Pruning{reasoning, confidence,
│                   mergedAreas[], mergedNeurons[]}
│
└── Nucleus.integrate(pruning, callback)        Ch.9,12
    └── [thread pool]
        ├── eliminate(pruning)
        │   ├── mergedAreas   -> sources -> removeAll()
        │   └── mergedNeurons -> sources -> removeAll()
        ├── consolidate(pruning)
        │   ├── mergedNeurons -> newNeuron -> store()
        │   └── mergedAreas   -> sanitize() + store()
        └── Episode.decay()
            └── keep CAPACITY_PER_AREA x areaCount


Transmitter Pipeline
=====================

Transmitter.transmit(Impulse, Class<T>)
├── PATHWAYS lookup
│   ├── Decision     -> Cortex
│   ├── Projection   -> Thalamus
│   ├── Fluctuation  -> Default
│   ├── Potentiation -> Plasticity
│   └── Pruning      -> Plasticity
├── Encoder.encode(impulse, caller)     -> prompt assembly
├── DiffusicService.call(signal)        -> Claude API
└── Decoder.decode(raw, response)       -> JSON parse + validate


Recursive Flow Summary
=======================

[R1] FIRE       -> Cortex.respond()     Effector result fed back for next decision
[R2] POTENTIATE -> Cortex.respond()     After neural growth, re-evaluate input
[R3] PROJECT    -> Cortex.respond()     Cross-area projection (Ch.20 association fibers)
[R4] DMN        -> Thalamus.relay()     Internal impulse enters CEN pathway
